# 进服需要装哪些模组/材质包？

## 问题描述
了解进入Minecraft服务器需要安装的模组和材质包要求。

## 纯净服要求

### 纯净服定义
- **无模组要求**：多数纯净服无需额外安装模组
- **自动材质包**：服务器会自动推送和下载必要材质包
- **版本匹配**：只需客户端版本与服务器版本匹配

### 纯净服进入
1. **检查版本**：确认客户端版本与服务器要求一致
2. **连接服务器**：直接输入服务器IP和端口连接
3. **接受材质包**：连接时接受服务器推送的材质包
4. **等待下载**：等待材质包下载完成进入游戏

## 模组服要求

### 常见模组类型
- **优化模组**：如OptiFine、Sodium等性能优化模组
- **功能模组**：如JourneyMap、Xaero's Minimap等地图模组
- **客户端模组**：如VoxelMap、Litematica等客户端辅助
- **服务器模组**：如特定游戏玩法的服务器专属模组

### 模组服进入流程
1. **查看服务器要求**：在服务器官网或论坛查看模组列表
2. **下载模组文件**：从官方或授权源下载必要模组
3. **安装模组**：将模组文件放入客户端mods文件夹
4. **启动游戏**：使用对应模组加载器（如Forge、Fabric）启动
5. **连接服务器**：输入服务器信息连接

## 材质包要求

### 服务器材质包
- **自动推送**：多数服务器会自动推送专属材质包
- **手动安装**：部分服务器需要手动下载安装材质包
- **版本兼容**：确保材质包与客户端版本兼容

### 材质包安装
1. **自动安装**：连接时接受服务器推送的材质包
2. **手动下载**：从服务器官网下载材质包文件
3. **放入文件夹**：将材质包放入resourcepacks文件夹
4. **游戏内启用**：在游戏设置中启用材质包

## 特殊服务器要求

### 大型模组包服务器
- **整合包**：可能需要下载完整整合包
- **启动器**：使用特定启动器（如CurseForge、MultiMC）
- **版本锁定**：严格版本锁定，不能随意更改模组

### 角色扮演服务器
- **专属模组**：可能需要服务器专属角色扮演模组
- **材质要求**：特定角色扮演材质包
- **语音模组**：如Vivox等语音聊天模组

### 小游戏服务器
- **大厅模组**：可能需要大厅或小游戏专属模组
- **反作弊**：部分服务器需要反作弊客户端模组
- **统计模组**：游戏统计和排名模组

## 常见问题处理

### 模组冲突
1. **检查兼容性**：确认模组间和与版本兼容
2. **逐一测试**：逐一启用模组测试冲突
3. **更新模组**：更新到最新兼容版本
4. **寻求帮助**：在服务器社区寻求模组冲突帮助

### 材质包不显示
1. **检查安装**：确认材质包正确安装和启用
2. **版本检查**：检查材质包与游戏版本匹配
3. **重新下载**：重新下载材质包文件
4. **联系管理**：联系管理员检查服务器推送

### 连接失败
1. **模组检查**：检查是否缺少必要模组
2. **版本验证**：验证客户端与服务器版本一致
3. **防火墙**：检查防火墙是否拦截连接
4. **服务器状态**：检查服务器是否在线

## 注意事项
- 只从官方或授权源下载模组和材质包
- 定期更新模组保持兼容性
- 备份重要存档和设置
- 遵守服务器模组使用规则
