# 无法连接服务器？一步步排查 (｡•́︿•̀｡)🔌

> 先别着急砸键盘，按照清单逐项核对，90%的问题都能自己解决。

## 🔍 常见原因速查

| 可能原因 | 快速判断方法 |
| :--- | :--- |
| **IP/端口填错** | 核对群公告或官网的正确地址 |
| **服务器维护/离线** | 查看 [服务器状态页](http://uptime.s.3q.hair/status/qy)，看对应线路的在线率 |
| **人数已满** | 提示“服务器已满”字样，换备用线路或等一会再试 |
| **版本不匹配** | 确认你的客户端版本在 **1.21.5x - 1.26.1x** 之间 |
| **你被 ban 了** | 提示“你已被此服务器封禁”，联系管理申诉 |
| **本地网络/防火墙** | 重启路由器、关闭防火墙试试；电脑开热点给手机测试 |

## 🛠️ 按顺序操作

1. **查服务器状态**：打开 [服务器状态页](http://uptime.s.3q.hair/status/qy)。如果显示对应线路 **0%** 或“故障”，说明服务器挂了，耐心等待恢复。
2. **核对版本**：确保你的游戏版本在支持范围内。用启动器切换到 1.21.5x - 1.26.1x 之间的正式版。
3. **核对地址**：重新输入一遍 IP，把端口号用英文冒号加上。别多空格。
4. **换备用线路**：去群公告里找备用 IP/域名，填进去。之前讲过的 `/nodeui` 节点切换在这里用不上（连不进服），但给你提供了同样思路——用别的入口。
5. **排除本地**：重启路由器，关 VPN，关防火墙。拿手机流量开热点让电脑连，能连上就说明是你家网络问题。
