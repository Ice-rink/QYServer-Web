# BehaviorLog 行为监控速查 (・ω・)🔎

> 记录一切，让熊孩子无处遁形。所有玩家行为都会被自动记录。

## 📋 监控项目清单
系统会记录以下所有关键行为：

| 类别 | 具体监控项 |
| :--- | :--- |
| **👤 进出与交流** | 玩家进服、玩家离开、玩家发送聊天消息、玩家使用命令 |
| **⛏️ 方块操作** | 玩家破坏方块、玩家放置方块、玩家与方块交互、玩家使用物品点击方块 |
| **📦 容器与物品** | 容器内容改变、玩家打开容器UI、漏斗送出物品、玩家丢出物品 |
| **💥 环境事件** | 方块被爆炸破坏、弹射物品击中方块 |

## 🛠️ 两种查询方式

### 方式一：游戏内速查（方块限定）
- **操作**：准心对准要查的方块，输入 `/qbi`
- **限制**：只能查方块记录，且仅能追溯近 5 小时内数据
- **适用场景**：刚被拆，急需原地快速确认是谁干的

### 方式二：QQ群获取完整日志（推荐）
- **操作**：在主群、二群或互通群中发送 `/getlog`
- **收获**：获取服务器全部日志文件（`.csv` 格式）
- **深度查询**：用 Excel 或 WPS 打开 `.csv` 文件，可按玩家名、时间、行为类型筛选筛选，精确锁定破坏过程
> 不会查CSV？小心计算机挂科！

## 💡 结合 iLand 领地的反拆工作流
1. **遇袭** → 速查：准心对准被毁方块输 `/qbi` 锁定嫌疑人+时间。
2. **取证** → 去群发 `/getlog`，下 CSV，筛出该玩家在此时段所有破坏/交互记录，截图留证。
3. **加固** → 打开 `/land` 领地管理面板，进权限配置，把陌生人所有权限关掉。
4. **举报** → 拿 CSV 截图+游戏截图，找管理举报。
