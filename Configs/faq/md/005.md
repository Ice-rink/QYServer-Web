# iLand 领地速通手册 ( ͡° ͜ʖ ͡°)✧🏡

> 融合了「别人拆我家怎么办」的硬核防拆需求，把 iLand 玩家指令榨干给你看。本指南基于 [AirLand Project 官方文档](https://pingguo114514.top/docs/iland/#/)。

## 🚀 30秒上手
1. **圈地**：拿木锄头，左键选点A，右键选点B，或者直接输 `/land new`。
2. **购买**：输 `/land buy`，系统会扣款并创建领地。
3. **管理**：站在领地里输 `/land`，弹出一站式 [领地管理器](https://pingguo114514.top/docs/iland/#/user/lmgr) 面板。

## 🕹️ 面板核心功能
面板打开后，记住这几个救命的按钮（详见 [领地管理器文档](https://pingguo114514.top/docs/iland/#/user/lmgr)）：

| 按钮 | 能干嘛 |
| :--- | :--- |
| **权限配置** | ⚠️ 防拆核心！控制陌生人是否能破坏/放置/开关/用容器。 |
| **领地信任** | 把基友 ID 加进来，他就能自由建造，但**改不了你的权限设置**。 |
| **领地传送** | 设定传送点：`/land tp set` 设脚下，删掉用 `/land tp rm`。 |
| **领地昵称** | 给家起个名（比如：糖果屋），提示更亲切。 |
| **领地备注** | 写一行欢迎语，陌生人走进来会自动看到，支持 `$visitor`（访客ID）和 `$n`（换行）。 |
| **重选范围** | 搬家扩地用的，多退少补，选完输 `/land ok`。 |
| **过户** | 把领地完全送人，送完你就不再是主人了。 |
| **删除领地** | 删了并退款，**数据不可恢复**，别手滑。 |

## 🛡️ 防拆配置实操（必做）
打开领地面板 → **权限配置**：

- **破坏/放置**：绝对关掉。
- **使用 (use)**：关掉可以防熊孩子按按钮、开门、踩踏板。
- **容器**：关掉别人就打不开你的箱子。
- *小技巧*：被信任的基友**不受这些限制**，所以只给熟人开信任。

## ⌨️ 常用指令速查
所有指令以 `/land` 开头。完整指令表见 [官方指令文档](https://pingguo114514.top/docs/iland/#/user/commands)。

| 指令 | 作用 |
| :--- | :--- |
| `/land` | 呼出领地 Home 页 |
| `/land gui` | 调出领地选择列表（领地内外都能用） |
| `/land tp` | 打开传送列表，飞去其他领地 |
| `/land tp set` | 把脚下设为领地传送点 |
| `/land tp rm` | 删掉脚下领地的传送点（变回默认） |
| `/land giveup` | 放弃未完成圈地 / 未付款的领地 |
| `/land set` | 手动设定领地的 A/B 点 |

## 🧠 聪明人小贴士
- **防拆事后再怎么查，也不如权限一开始就关好。** 圈完地后第一件事，去权限里把所有陌生人开关拉满到“拒绝”。
- 昵称可以设成只有你懂的名字，便于一眼找到家；备注设成公共的，方便来访朋友知道规矩。
- 如果被信任的玩家搞事，**直接在面板里取消信任**，无需删领地。
