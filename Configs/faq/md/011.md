# 红石机器与刷怪塔建造指南 (˵ •̀ ᴗ - ˵ )⚡️

## 核心规则速览
- **✅ 允许**：基础电路、自动农场（作物/动物/村民）、物品分类与运输、装饰性红石。
- **⛔ 禁止**：高频电路、导致服务器明显卡顿的任何装置。
- **📍 位置**：通常限定在个人领地内建造。

## 性能评估清单
违反性能红线是主要的违规风险，管理会用以下维度做评估：

| 评估维度 | 红石机器风险点 | 刷怪塔风险点 |
| :--- | :--- | :--- |
| **更新频率** | 高频率红石更新 | / |
| **实体数量** | 生成的实体总量 | 堆叠的怪物数量 |
| **区块负载** | 影响的常加载区块数 | 影响的常加载区块数 |
| **Tick 占用** | 整体红石运算耗时 | 实体 AI 与寻路计算 |
| **掉落物** | / | 未及时收集的掉落物实体 |

## 最佳优化实践

### 🔴 红石机器
- **淘汰时钟**：优先用观察者，回避高频时钟电路。
- **精简电路**：减少不必要的红石粉更新，缩小影响面。
- **区块策略**：将机器主体锁定在常加载区块内，避免跨区块运行。
- **定期维保**：定期检修，防止意外高负载。

### 👾 刷怪塔
- **限流设计**：用开关或机制主动控制存活怪物上限。
- **即杀即收**：确保怪物迅速被处死，物品被漏斗瞬间吸收。
- **光照严控**：精确控制塔内光照，只在你想要的地方刷怪。
- **定时清理**：定期清剿因卡位残留的实体，防止累积。

## 违规处理梯度

| 等级 | 触发条件 | 处理方式 |
| :--- | :--- | :--- |
| **轻度** | 首次造成轻度性能波动 | 警告 + 限期修改 |
| **中度** | 重复违规或中度性能影响 | 罚款/没收、机器可能被重置或临时封禁 |
| **严重** | 故意违规或造成严重卡顿 | 永久封禁、强制拆除、经济处罚 |
