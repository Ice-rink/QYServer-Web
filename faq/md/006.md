# 被拆家应急与举报指南 (´;ω;`)🔍

## 🚨 60秒紧急止损
不要慌，按顺序来：

1. **截图锁证**：按 F2（或截图键），多角度拍下破坏现场、破坏者 ID、右上角时间戳。
2. **勿清现场**：别急着修补！保持被破坏的原样，方便管理取证。
4. **就地保护**：若还有未损部分，立即用指令或圈地工具保护剩余建筑。

## 🔎 证据收集清单
举报时，把这些一起提交，处理速度会快很多。

- ✅ **破坏者信息**：清晰的玩家 ID 截图（F3+B 开碰撞箱可显示得更清楚）。
- ✅ **案发现场**：被破坏前（如果有）、破坏后的对比截图，包含坐标（F3 调试界面）。
- ✅ **时间线索**：具体发生时间，精确到小时。领地页面的日志也可截图。
- ✅ **相关对话**：如有争吵或对方炫耀的聊天记录，一并截取。

## 📢 举报通道

- 打开服务器的 QQ 群。
- 将**图片证据 + 文字说明**打包发送。
- AT管理或腐竹，或直接私聊她们。


## 🏗️ 建筑恢复路径

| 你的情况 | 恢复方案 |
| :--- | :--- |
| **大规模方块被拆** | 提供建筑备份或请求**建筑回档**，管理视情况处理。 |
| **仅个别方块被毁** | 管理核实后可能会直接复原，或指导你手动补上，没收破坏者对应赔偿。 |

## 🛡️ 防拆指南（下次一定）
- **圈地要快**：资源攒够第一件事，就是圈领地，并禁掉陌生人的破坏、开关、容器权限。
- **保护好门窗**：门窗要有防火、隔音、锁扣，防止火灾、爆炸。
- **远离是非地**：不在出生点或公共资源区附近建价值太高的裸露建筑。

> 被拆不是你的错。保持冷静，证据齐全，管理一定会帮你~ (๑•̀ㅂ•́)و✧
