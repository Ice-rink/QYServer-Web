# 服务器禁止行为清单 (｡•̀ᴗ-)✧⚖️

## 核心红线
触碰以下任一条都会面临处罚，务必避开。

### 🚫 作弊与脚本
- **非法透视**：禁止 X-ray 材质包。
- **一切外挂**：战斗、移动、采集类作弊软件均禁止。
- **非正常移动**：禁止未授权的飞行、加速、无跌落水上漂等。

### 🚫 恶意破坏
- **物理破坏**：严禁拆家、毁坏他人建筑、领地及公共设施。
- **财产侵犯**：严禁偷窃箱子、展示框等容器内物品。
- **领地入侵**：未获许可，禁止在他人领地内破坏。

### 🚫 骚扰与言论
- **人身攻击**：辱骂、歧视、引战、阴阳怪气均不允许。
- **恶意刷屏**：连续发重复消息、无意义字符或广告。
- **威胁勒索**：任何形式的威胁、恐吓、索要财物行为。
- **隐私侵犯**：公开他人真实信息、照片或私下对话记录即算。

## 机制滥用
属于钻空子的“高智商”违规，管理看得见。

| 类别 | 禁止行为 | 为什么不行 |
| :--- | :--- | :--- |
| **经济滥用** | 物品复制、刷钱、恶意垄断资源、欺诈交易 | 破坏全员游戏体验与经济平衡 |
| **红石滥用** | 卡服机、高频电路、无限刷怪塔、卡区块装置 | 服务器 TPS 杀手，一人卡全服 |
| **漏洞利用** | 复制物品/权限、利用插件 Bug | 发现漏洞不报还利用，重罚 |

## 社区与账号安全

- **账号规范**：禁止绕过封禁，禁止冒充管理员或名人。
- **社区秩序**：禁止在群内引战、传播不实信息、恶意举报浪费管理时间。
- **网络安全**：严禁 DDoS/CC 攻击、尝试破解他人账号、非法获取服务器文件或后台权限。
- **权限滥用**：拥有特殊权限的玩家若包庇、滥用指令或干扰插件运行，一律从重处理。

## 违规处理梯度

| 等级 | 典型行为 | 罚则 |
| :--- | :--- | :--- |
| **轻度** | 初犯口角、无恶意无意破坏、小范围刷屏 | 警告 / 临时禁言 |
| **中度** | 小号作弊、恶意破坏、重复骚扰、经济欺诈 | 临时封禁 / 没收财产 / 建筑回滚 |
| **严重** | 外挂、复制物品、攻击服务器、冒用身份 | **永久封禁** + **全服通报** |
